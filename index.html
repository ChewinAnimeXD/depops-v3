DepOps en Github por Andrea Salcedo y Edwin Sanabria

DepOps es una metodología que tiene como objetivo integrar el proceso de desarrollo de software con las operaciones del sistema. Esta extensión de DevOps se enfoca específicamente en la automatización del despliegue de aplicaciones y en la gestión de la infraestructura necesaria para ejecutarlas de manera eficiente.

Con github action podemos automatizar las herramientas de DepOps de manera muy facil
